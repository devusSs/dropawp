# dropawp

dropawp is a Go tool to track your [Counter-Strike](https://www.counter-strike.net/) inventory value over time.

It is designed to run fully client sided and can be used in any interval desired by the user.

## Usage

Simply run `dropawp` to get an overview over all the commands and subcommands as well as available flags.

Sample output of running `dropawp` (as of 26th June 2025):

```
A Go tool to track your Counter-Strike 2 inventory.

Usage:
  dropawp [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command
  projects    Add, list and delete tracking projects
  update      Check and apply available updates
  version     Prints the build information of the application and exits.

Flags:
  -h, --help   help for dropawp

Use "dropawp [command] --help" for more information about a command.
```
